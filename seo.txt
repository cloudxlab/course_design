4. STRUCTURED DATA / SCHEMA MARKUP (Missing — high impact)
-------------------------------------------------------------

Adding JSON-LD structured data helps Google show rich results
(star ratings, course details, price, dates) directly in search.

Add a <script type="application/ld+json"> block in <head> with
the following schemas:

  a) Course Schema
  {
    "@context": "https://schema.org",
    "@type": "Course",
    "name": "The Inventor's Journey: From First Principles to AI",
    "description": "8-month weekend program teaching Python, ML, Deep Learning and AI by invention. IIT Roorkee certificate included.",
    "provider": {
      "@type": "Organization",
      "name": "CloudxLab",
      "sameAs": "https://cloudxlab.com"
    },
    "educationalLevel": "Beginner to Advanced",
    "timeRequired": "P8M",
    "teaches": ["Python", "Machine Learning", "Deep Learning", "AI Agents", "LLMs", "Reinforcement Learning"],
    "hasCourseInstance": {
      "@type": "CourseInstance",
      "courseMode": "online",
      "duration": "P8M",
      "instructor": {
        "@type": "Person",
        "name": "Sandeep Giri"
      }
    }
  }

  b) Organization Schema for CloudxLab
  c) BreadcrumbList if the page is part of a larger site

8. PAGE SPEED / CORE WEB VITALS
-------------------------------------------------------------

  a) Images: All background images (hero_bg.jpg, campus_bg.jpg etc.)
     are large JPEGs. Convert to WebP format — typically 30–50% smaller
     with same quality. Use <picture> elements with WebP + JPEG fallback
     for inline images.

  b) Google Fonts: The page loads two font families from Google with
     9 weights. Add preconnect hints:
       <link rel="preconnect" href="https://fonts.googleapis.com" />
       <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
     (These are missing — the browser must do an extra DNS lookup first.)

  c) Font display: Add &display=swap to the Google Fonts URL
     (already present ✅).

  d) Lazy loading: Inline images use loading="lazy" ✅.
     Consider adding fetchpriority="high" to the hero background or
     a visible LCP image.

  e) CSS is inline in <style> — good for a single-page site (no extra
     request), but consider critical CSS separation if the page grows.
